# Business Calendar

Современное мобильное приложение для эффективного управления временем и деловыми задачами.

## Стек технологий
- **Framework:** Flutter
- **Architecture:** Clean Architecture + Modular approach
- **Language:** Dart (Null Safety)

## Текущий статус
Проект находится в стадии начальной разработки. Создан базовый каркас (skeleton) с настроенной навигацией, темами и заготовками для модулей:
- Авторизация
- Календарь
- Заметки
- Контакты
- Ассистенты
- Подписки

## Как запустить
1. Склонируйте репозиторий.
2. Выполните `flutter pub get`.
3. Запустите проект через `flutter run`.

## Структура проекта
- `lib/app`: Ядро приложения (роутинг, темы).
- `lib/core`: Общие сервисы, модели и ошибки.
- `lib/features`: Модули функционала.
- `lib/shared`: Переиспользуемые виджеты и утилиты.
